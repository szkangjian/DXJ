## 1. 产品概要：EWY 是什么

| 项目 | 内容 |
| ---- | ---- |
| 全称 | iShares MSCI South Korea ETF |
| 发行方 | BlackRock (iShares) |
| 交易所 | NYSE Arca |
| 日均成交额 | ~$3.9 亿 |
| Spread | ~$0.02（0.015%） |
| 年化波动率 | 29%（均值），当前 ~70% |
| 股息 | 季度，年化 ~1.5%（不可做 Dividend Capture） |
| 前三大持仓 | 三星电子 ~23%、SK Hynix ~18%、三星 SDI ~3% |
| 核心驱动 | 半导体/AI 周期（~41%）+ 地缘政治（伊朗/霍尔木兹） |

**EWY 本质上是一只半导体 ETF**。三星 + SK Hynix 占比 >40%，价格走势主要由 HBM/DRAM 需求周期和地缘政治冲击驱动。

### 与 STRC 的本质区别

| 维度 | STRC | EWY |
|------|------|-----|
| 价格锚定 | $100 面值（ATM 增发硬顶） | 无锚定，趋势性强 |
| 股息 | 月度 ~$0.96（ROC 免税） | 季度 ~$0.50（普通股息） |
| 策略基础 | 围绕面值的套利 | 大跌后的短期反弹 |
| 持有逻辑 | 时间站在你这边（每月收息） | 时间是风险（无股息托底） |
| 杠杆 | 可用（有股息覆盖利息） | **不可用** |

## 2. 策略架构：双策略并行

```text
┌─────────────────────────────────────────┐
│     主策略：IBS 均值回归（收盘信号）      │
│  → IBS < 0.2 且 Close > MA200 → 买入     │
│  → IBS > 0.8 → 卖出                     │
│  → 83% 胜率，均收益 +2.14%/笔            │
│  → 平均持有 2.3 天                       │
├─────────────────────────────────────────┤
│   辅助策略：跌幅触发（盘中信号）          │
│  → 盘中跌破前日收盘 -3% → 买入           │
│  → 反弹 +2.5% 或持有 3 天 → 卖出         │
│  → 73% 胜率，捕捉极端行情                │
└─────────────────────────────────────────┘
```

两个策略信号时机不同（收盘 vs 盘中），可以并行运行。

## 3. 主策略：IBS 均值回归

### 3.1 IBS 是什么

```
IBS = (Close - Low) / (High - Low)
```

- IBS < 0.2 = 收盘贴近最低点 → 空方全天压制，超卖
- IBS > 0.8 = 收盘贴近最高点 → 多方全天主导，该走了

### 3.2 买入规则

| 条件 | 要求 |
|------|------|
| IBS | < 0.2 |
| 价格 | Close > 200 日均线 |
| 熔断检查 | 无活跃熔断条件（见第 6 节） |

**执行时机**：收盘前 5 分钟（3:55 PM ET），确认 IBS 即将 < 0.2 后下单。
**下单方式**：MOC（Market on Close）或 3:58 PM 市价单。

### 3.3 卖出规则

| 条件 | 动作 |
|------|------|
| 当日 IBS > 0.8 | 收盘卖出 |
| 持有满 10 个交易日 | 到期平仓（安全阀） |

**执行时机**：收盘前 5 分钟确认 IBS > 0.8 后下单。

### 3.4 回测表现

| 指标 | 数值 |
|------|------|
| 交易笔数 | 29（10 个月） |
| 胜率 | 83% |
| 总收益 | +62.0% |
| 均收益/笔 | +2.14% |
| 最大单笔亏损 | -3.00% |
| 平均持有 | 2.3 天 |
| 退出方式 | 29/29 笔全部 IBS > 0.8 正常退出 |

### 3.5 关键认知

- **不需要判断事件类型** — 无论什么原因导致的低 IBS，次日反弹概率都很高
- **不设单笔止损** — EWY 经常先跌后反弹，止损会在最差价格卖出
- **MA200 当前不起作用** — EWY 远在均线之上，但留着做未来趋势反转的自动保护

## 4. 辅助策略：盘中跌幅触发

### 4.1 买入规则

| 条件 | 要求 |
|------|------|
| 盘中跌幅 | 相对前日收盘 < -3% |
| MA200 | 当前价 > 200 日均线 |
| 熔断检查 | 无活跃熔断条件 |

**执行时机**：盘中任何时刻触发即买入（限价单在 -3% 位置挂好）。

### 4.2 卖出规则

| 条件 | 动作 |
|------|------|
| 从买入价反弹 +2.5% | 达标卖出 |
| 持有满 3 个交易日 | 到期平仓 |

### 4.3 连续触发（网格效应）

如果买入后继续下跌并再次触发 -3%（相对新的前日收盘），**应该买入**。这是一个全新的交易信号。

- 到期卖出和新信号买入是两个**独立判断**
- 回测中连续入场（如 2026 年 3 月伊朗战争）整体正收益
- 第一笔可能亏，但后续笔在更低价位入场，反弹概率更高

### 4.4 什么时候不买

- 同一日已有 IBS 策略持仓 → 仍然买入（两个策略独立运行）
- 当日触发 -3% 但收盘 IBS > 0.5 → 仍然按跌幅策略操作（IBS 是独立信号）

## 5. 仓位管理

### 5.1 资金分配

| 用途 | 比例 | 说明 |
|------|------|------|
| IBS 策略 | 每笔投入 **固定金额** | 例：$20,000/笔 |
| 跌幅策略 | 每笔投入 **固定金额** | 例：$20,000/笔 |
| 现金储备 | 始终保留 **30%** | 应对连续触发 + 日常流动性 |

### 5.2 最大同时持仓

- IBS 策略：**1 笔**（不会同时有两个 IBS 信号）
- 跌幅策略：**最多 3 笔**（连续下跌时网格入场）
- 合计最大敞口：**4 × 固定金额**

### 5.3 绝不用杠杆

EWY 没有 STRC 的股息托底。杠杆 + 波动率 70% = 爆仓风险。**现金账户操作。**

## 6. 风控铁律

### 6.1 不设单笔止损

| 原因 | 数据支持 |
|------|---------|
| EWY 大跌后经常先继续跌再 V 反 | 回测：无止损 +22.6% vs -2%止损 +12.0% |
| 止损后往往在最差价格卖出 | 2026-03 伊朗战争期间止损反而亏更多 |
| 继续下跌会触发新的买入信号 | 网格效应自动在更低价位加仓 |

### 6.2 策略级熔断（必须执行）

| 熔断条件 | 触发动作 | 恢复条件 |
|---------|---------|---------|
| **连续 3 笔到期亏损** | 暂停所有策略，不接新信号 | 人工复盘确认后恢复 |
| **EWY 跌破 200 日均线** | 暂停所有策略 | 重新站上 MA200 + 10 个交易日 |

> 回测 2 年内从未出现连续 3 笔到期亏损。触发了说明市场性质已变。

### 6.3 事件级否决权（极少使用）

以下**明确的基本面断裂**可否决信号：

| 事件 | 行动 |
|------|------|
| 三星/SK Hynix 季报业绩暴雷 | 暂停至下季财报 |
| 美国对韩半导体新制裁 | 暂停至政策明朗 |
| 韩国发生政变/战争 | 暂停至局势稳定 |

以下**不否决**：
- "我觉得还会跌" ❌
- "伊朗局势在恶化" ❌
- "大盘看起来很弱" ❌

回测证明：伊朗战争期间严格执行，每笔都赚。**模糊判断不否决，只有确定性的基本面断裂才否决。**

| NVIDIA 季度指引 | 财报 | 下季指引低于预期 |

## 8. 税务速览

EWY 作为美股 ETF，税务结构简单。详细分析见 [04_ewy_tax_analysis.md](04_ewy_tax_analysis.md)。

| 维度 | 美国居民 | NRA (非美居民) |
|------|---------|---------------|
| IBS 策略收益（短期 CG） | 并入普通收入（最高 37%） | **零美国税** |
| EWY 季度股息 | 预扣 30% → 报税时用 FTC 抵扣 | 预扣 30%（或按税收协定 10%） |
| Wash Sale | 适用（30 天内同标的买卖） | **不适用** |
| 申报要求 | Schedule D + Form 8949 | 保持 W-8BEN 有效即可 |

### NRA 优势

IBS 策略持有 1-3 天，全部是短期资本利得。NRA **零美国资本利得税**，策略收益 100% 落袋。

### Wash Sale 注意（美国居民）

IBS 策略频繁买卖 EWY，30 天内的亏损交易会被 Wash Sale 规则递延。建议：
- 让 IB 自动追踪 Wash Sale（默认开启）
- 每年 12 月检查累计递延金额
- 如需实现亏损：年底停止交易 31 天

## 9. 自动化工具链

实时监控：
- 盘中跌幅是否触发 -3% 买入信号
- 持仓是否达到 +2.5% 反弹目标
