## 六、第一轮观察

- 静态相关性最高的候选因子是 `EWJ`，需要优先作为主驱动假设验证。
- DXJ 与日本股市代理的联动强于与美国大盘的联动，应先从日本 beta 和日本主题驱动解释。
- 日元代理与 DXJ 存在可见联动，后续应单独测试汇率环境对策略表现的影响。

这些结论仍是驱动映射层，不是交易结论。
下一步应把 DXJ 的异动日逐一分类，分清哪些是日本 beta、哪些是汇率窗口、哪些是全球风险事件。
